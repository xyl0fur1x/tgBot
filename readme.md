# A telegram-based Bot for creating an online shop

## usage steps:
### 1. npm install
### 2. provide an API_URL inside .env with 3000 port
### 3. provide a token for your bot inside .env
### 4. Inside main.ts, provide your Telegram ID and Username inside "botDevs" and "manager" consts. You can get it via this bot, using /getme command, or simply use any other you may sknow
### 5. npm run serverScript
### 6. npm run botScript



## By default, it compiles to the test_dist folder, you can change it inside .tsbuildinfo later
